
# Challenges

## Overview

The "challenges" project is a collection of JavaScript challenges and solutions. Each challenge is solved in a separate JavaScript file.

- [Installation](#installation)
- [Usage](#usage)
- [Scripts](#scripts)

## Installation

To use this project, you will need [Node.js](https://nodejs.org/) installed on your computer.

1. Clone this repository to your local machine:

   ```
   git clone https://github.com/HolisticDevelop/Kata.git
   ```

2. Navigate to the project directory:

   ```
   cd Kata
   ```

3. Install the project dependencies:

   ```
   npm install
   ```

## Usage

This project is designed to showcase solutions to various coding challenges. You can run the challenges by executing the provided scripts.

## Scripts

The following scripts are available:

- `npm start`: Executes the default challenge (ch1.js).
- `npm run ch1`: Executes challenge 1 (ch1.js).
- `npm run ch2`: Executes challenge 2 (ch2.js).
- `npm run ch3`: Executes challenge 3 (ch3.js).
